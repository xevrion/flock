## What is Flock?

Flock is an open-source SDK that adds real-time multiplayer presence to any web app — live cursors, who's online, selection indicators — in under 10 lines of code.

It has three parts:
- **`@flock-sdk/react`** — React hooks for cursors, presence, and connection status
- **`@flock-sdk/core`** — Framework-agnostic client (vanilla JS, other frameworks)
- **`@flock-sdk/server`** — Self-hostable WebSocket server (Docker or npx)
